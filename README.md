## üèóÔ∏è System Architecture & Scalability

The architecture is designed as a **Distributed Node Network**. Each bin operates as a unique node (e.g., `ECO-DXB-MAIN-01`), allowing for city-wide deployment.

1. **Detection:** Ultra-fast YOLO11 identifies the item (Can vs. Cup).
2. **Validation:** AI checks for contamination. If valid, the "Green Credit" is calculated.
3. **Hashing:** A unique block hash is generated, linking the timestamp, item type, and user points.
4. **Action:** The user is directed to the correct bin (Blue/Green/Brown) via the dashboard.
